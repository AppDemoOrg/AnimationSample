# AnimationSample    

## 动画分类   
1、Tween（补间）动画包括：平移、旋转、缩放、透明度，View动画是一种渐变式动画      
2、帧动画：图片切换动画       
3、属性动画：通过动态改变对象的属性达到动画效果
